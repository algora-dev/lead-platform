# Roadmap

## Phase 1 — reliable internal tool
- Complete Windows setup and run UK/NZ scans.
- Add tests for scoring, URL canonicalisation and company normalisation.
- Improve company matching and contact validation.
- Add advert expiry and rescan behaviour.

## Phase 2 — configurable demos
- Dynamic dashboard columns and labels from tenant fields.
- CSV upload into the same record normaliser and score engine.
- Tenant creation wizard or questionnaire-to-config script.
- Demo mode with capped scans and isolated data.

## Phase 3 — commercial workflow
- Authentication, tenant isolation and user roles.
- Saved views, assignments, notes and audit history.
- Export and CRM integrations.
- Email templates, consent/suppression controls and queued sending.

## Phase 4 — optional channels
- SMS provider integration.
- AI-assisted call preparation and, where lawful, automated calling.
- Campaign analytics and follow-up automation.
